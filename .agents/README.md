# Project Agent Memory

This directory is the permanent source of truth for project context, architecture, decisions, work progress, issues, and change history.

Before modifying the application, future developers and AI agents must read every relevant document here, beginning with `PROJECT_CONTEXT.md`, `ARCHITECTURE.md`, `TODO.md`, `DECISIONS.md`, `ISSUES.md`, and the matching file under `modules/`. Update `TODO.md` and `WORK_LOG.md` before coding, and finish every verified task by updating `CHANGELOG.md`, `WORK_LOG.md`, and any affected context documents.

Historical entries must be preserved. Confirmed facts and assumptions must be clearly distinguished.
